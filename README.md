# UiCraft 🎨

UiCraft is a community-driven platform providing free open-source UI components, design inspiration, frontend resources, and complete webpage templates for developers. Built with a focus on modern aesthetics—including high-contrast Neo-Brutalism and sleek minimalism—UiCraft helps you build stunning user interfaces faster without the overhead of heavy frameworks.

---

## Features ✨

*   **Zero Dependencies:** Pure HTML, CSS, and Vanilla JavaScript. Easy to copy, paste, and adapt into any tech stack (React, Vue, Angular, Next.js, or static sites).
*   **Neo-Brutalism & Modern UI:** Features high-contrast borders, solid offset drop shadows, interactive hover physics, and premium typography (Inter).
*   **Dark Mode Ready 🌓:** Out of the box CSS variable support for light and dark themes using a simple `data-theme` attribute.
*   **Massive Component Library 🧩:** Includes UI Buttons, Modern Forms, Component Cards, Hero Sections, Navigation Menus, Dashboard UIs, Pricing Tables, Loading Animations, Data Tables, and Modals.
*   **Full Page Templates 🚀:** Production-ready layout templates for AI Agencies, Bakeries, Cloud Management SaaS, E-commerce, and Portfolios.

---

## Tech Stack 🛠️

*   **Markup:** HTML5
*   **Styling:** Vanilla CSS (CSS Variables, Flexbox, CSS Grid)
*   **Interactivity:** Vanilla JavaScript
*   **Typography & Icons:** FontAwesome (Icons) & Google Fonts (Inter, Space Mono)

## Getting Started 🚀

Because UiCraft is built with zero dependencies, using it is incredibly simple. You don't need to install any packages!

### 1. Clone the Repository
Open your terminal and run:
```bash
git clone https://github.com/thecodecraftdev/uicraft.git
cd uicraft
```

### 2. Explore the Library
Simply open `index.html` or `ui-ux-webpages.html` in your favorite web browser to browse the full component and template library.

### 3. Using Components
1. Navigate to the component category you need (e.g., `modern-forms.html` or `ui-buttons.html`).
2. Find the design you like.
3. Click "Get Code" (or copy the HTML/CSS directly from the source code).
4. Paste it into your project!

## Theming Guide 🎨

UiCraft relies heavily on CSS variables for rapid theme switching. To customize the colors for your project, simply override the root variables:

```css
:root {
    --bg-color: #f4f0ea;
    --text-color: #111111;
    --border-color: #111111;
    --shadow-color: #111111;
    --primary: #ffd836;    /* Your brand color */
    --secondary: #ff6b6b;  /* Accent color */
    --tertiary: #48dbfb;
    --card-bg: #ffffff;
    --nav-bg: #f4f0ea;
}
```
Toggle dark mode by setting `<html data-theme="dark">` on your root element.
